## pip

- `pip install <package>`: 패키지 설치
- `pip install "package~=3.0.0"`: 3.0.0 버전의 패키지 설치
- `pip uninstall <package>`: 패키지 삭제
- `pip freeze`: 설치된 패키지 확인
- `pip freeze > requirements.txt`: requirements.txt 파일에 설치된 패키지 출력
- `pip install -r requirements.txt`: requirements.txt 파일에 기록된 패키지를 설치

## venv

- `source .venv/Scripts/activate`: 가상환경 실행
- `deactivate`: 가상환경 끄기
